# 安鸿飞毕业设计前端
## 题目：基于React疫情数据统计，分析和可视化得实现
## 使用技术：
>1. typescript
>2. react
>3. style-components
>4. mobx
>5. bizcharts
>6. wangEditor
## 实现功能：
>1. 用户登陆
>2. 注册
>3. 信息类操作
>4. 评论操作
>5. 疫情数据大屏
>6. 疫情数据分析

**Tips: 
用户分为两种：1.普通用户；2.管理员
*desc*:1.普通用户只能操作自己的信息和评论
      2.管理员可以操作所有人得信息和评论**

# 项目操作：
## 1.安装项目依赖
`yarn`
## 2.运行项目
`yarn start`
## 3.项目打包
`yarn build`
